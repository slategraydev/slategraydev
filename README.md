Modified version of the PacMan commit repo, re-wrote portions of it to support custom mazes.
The ghosts now use their original code and PacMan uses A* search. 
I've also tossed in some training for PacMan, each night 100 iterations are run and the highest scoring variant gets output as the SVG.
